# 「スケートボードで」あり文の grammatical 到達確認レポート

更新日: 2026-02-27  
対象環境: Perl原本 (`syncsemphone.cgi`)  
対象文法: `imi01`  
対象 Numeration: `imi01/set-numeration/1606324760.num`  
観察文: 「ジョンがメアリをスケートボードで追いかけた」

## 1. 結論
「スケートボードで」がある場合でも、適切な規則適用順を取れば `grammatical` な状態に到達可能である。

## 2. 検証条件
1. Perl原本ページを開く  
   `http://127.0.0.1:8000/v1/legacy/perl/syncsemphone.cgi?grammar=3&mode=numeration_select`
2. `[ジョンがメアリをスケートボードで追いかけた] 1606324760.num` を選択
3. `numeration` -> `ok` を押して Grammar 適用画面へ進む

## 3. 到達手順（実操作で確認済み）
以下の順に 7 手を適用する。

1. `([x7-1 x8-1] LH-Merge)`  
   述語幹と時制を統合
2. `([x5-1 x6-1] RH-Merge)`  
   「スケートボードで」を形成
3. `([x1-1 x2-1] LH-Merge)`  
   主語句を形成
4. `([x3-1 x4-1] LH-Merge)`  
   目的語句を形成
5. `([x3-1 x7-1] RH-Merge)`  
   目的語句を述語へ統合（Theme 解消）
6. `([x6-1 x7-1] RH-Merge)`  
   道具句を述語へ統合（Instrument 解消）
7. `([x1-1 x7-1] RH-Merge)`  
   主語句を述語へ統合（Agent 解消）

## 4. 確認結果
7手目の適用後、Perl画面で次が表示されることを確認した。

- `解釈不可能素性がなくなったので、この表示は grammatical です。`

また、下部 process 履歴の先頭行は次と一致した。

`([x7-1 x8-1] LH-Merge) ([x5-1 x6-1] RH-Merge) ([x1-1 x2-1] LH-Merge) ([x3-1 x4-1] LH-Merge) ([x3-1 x7-1] RH-Merge) ([x6-1 x7-1] RH-Merge) ([x1-1 x7-1] RH-Merge)`

## 5. 成功例との差分整理
成功例「ジョンがメアリを追いかけた」と比べると、失敗例側には次の 2 語彙が追加される。

- `x5-1 N`（スケートボード）
- `x6-1 Z`（で）

この追加により、`x6-1` の未解釈素性（`+V(left)(nonhead)`）を解消するための統合手順が 1 ブロック増える。  
そのため、成功例の骨格（主語句・目的語句・述語句の統合）に加えて、道具句統合（手順 2 と 6）を入れることで `grammatical` に到達する。
